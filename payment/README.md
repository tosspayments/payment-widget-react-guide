# 토스페이먼츠 결제창 SDK

[토스페이먼츠 결재창 SDK](https://docs.tosspayments.com/sdk/payment-js) 샘플 프로젝트입니다.

- `/payment-window`: 통합결제창 연동 샘플

- `/payment-direct-window`: 카드 자체창 연동 샘플

- `/payment-easypay-window`: 간편결제 자체창 연동 샘플
