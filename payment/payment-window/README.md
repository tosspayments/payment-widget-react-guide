# 결제창 연동 샘플

토스페이먼츠의 결제창을 열 수 있는 연동 샘플입니다. 각 폴더에서 언어별 샘플을 확인할 수 있습니다.

카드 및 간편결제(통합결제창), 가상게좌, 계좌이체, 휴대폰, 상품권, 토스페이 결제창을 띄울 수 있습니다.
